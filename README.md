# Inbox Shield
InboxShield is an automated email security application designed to intercept, analyze, and isolate phishing threats before they reach an end-user’s inbox. By evaluating inbound email metadata, verifying domain authenticity, and deeply inspecting embedded URLs for hidden redirects or IP logging scripts, the system provides real-time defense against modern digital social engineering tactics.

# Purpose
Phishing remains the leading vector for digital crime, credential theft, and malware deployment. Attackers frequently exploit gaps in standard email filters using advanced evasion techniques like URL shortening, multi-stage HTTP redirects, and lookalike domains. The purpose of this project is to research these current evasion mechanisms and develop a prototype automation tool capable of unmasking and neutralizing these specific high-risk elements.
